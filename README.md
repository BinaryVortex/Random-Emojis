# Random-Emojis

A tiny, playful web app that generates random emoji combinations. Built with plain HTML, CSS, and JavaScript — no build tools or frameworks required.

## Features

- Generates random emoji combinations on demand
- Simple, dependency-free frontend (HTML/CSS/JS)
- Lightweight and easy to customize

## Usage

1. Clone the repository:

```bash
git clone https://github.com/BinaryVortex/Random-Emojis.git
cd Random-Emojis
```

2. Open `index.html` in your browser (double-click or use a local static server):

```bash
# Using Python 3's simple HTTP server (optional)
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

3. Click the button (or reload) to generate new random emoji combinations.

## Customize

- Edit the emoji list in the JavaScript file to add or remove emojis.
- Tweak the styles in the CSS file to change layout, colors, or animation.
